# Abemaciclib: 유방암에서 류마티스 관절염으로

## 한 문장 요약

Abemaciclib은 CDK4/6(사이클린 의존성 인산화효소 4/6)을 선택적으로 억제하는 표적항암제로, 글로벌 시장에서 호르몬 수용체 양성(HR+)/HER2 음성(-) 유방암 치료에 사용됩니다.
TxGNN 모델은 **류마티스 관절염(Rheumatoid Arthritis)**에 효과가 있을 수 있다고 예측하며,
현재 관련 임상시험 등록은 없고 **1편의 간접 문헌**만이 이 방향을 부분적으로 지지합니다.

## 이 예측이 타당한 이유는?

Abemaciclib은 세포주기 G1/S 전환의 핵심 조절자인 CDK4와 CDK6를 선택적으로 억제하여 암세포의 무제한 증식을 차단하는 소분자 표적치료제입니다. 현재 상세 작용 기전(MOA) 데이터는 DrugBank API 조회를 통한 보완이 필요한 상태이지만, 글로벌 Phase 3 임상(MONARCH 2, monarchE 등)을 통해 HR+/HER2- 유방암에서의 효능이 강력히 입증되어 있습니다.

CDK4/6 억제제는 항종양 효과 외에도 면역조절 잠재력을 보유합니다. CDK6는 T세포의 활성화 및 증식 조절에 직접 관여하며, CDK4/6 억제는 Th17/Treg 면역 균형에 영향을 주고 IL-6, TNF-α 등 핵심 염증성 사이토카인 분비를 조절할 가능성이 있습니다. 류마티스 관절염의 병태생리(자가반응성 T세포에 의한 관절 염증)와 기전적 교차점이 이론적으로 존재합니다.

그러나 현재 유일한 관련 문헌(PMID 40504547)은 유방암 환자에서 CDK4/6 억제제 투여 중 자가면역 질환이 발생한 사례를 조사한 약물경계 관찰 연구로, 이는 치료 효능 연구가 아닌 **안전성 신호** 연구입니다. 류마티스 관절염에 대한 재창출 근거는 현재 기전 추론 수준에 머물러 있어 추가 전임상 검증이 필수적입니다.

## 결론 및 다음 단계

**결정: Hold**

**사유:**
류마티스 관절염에 대한 Abemaciclib의 치료 효능을 지지하는 임상시험이 전혀 없으며, 유일한 관련 문헌도 치료 효능이 아닌 안전성 신호 연구입니다. 기전적 연결고리(CDK6-T세포 조절)는 이론적 가능성 수준에 불과하며 독립적인 전임상 검증이 선행되어야 합니다.

**진행하려면 필요한 것:**
- CDK4/6 억제와 Th17/Treg 균형에 관한 전임상 연구 (관절염 세포·동물 모델)
- Abemaciclib의 상세 작용 기전 데이터 (DrugBank API 조회 필요, DG002 해소)
- 한국 식약처(MFDS) 허가사항 및 경고문 확인 (현재 미허가 품목, DG001 해소)
- 자가면역 질환 맥락에서의 면역조절 효과 직접 탐색 연구
